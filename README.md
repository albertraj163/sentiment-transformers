# Sentiment Analysis using Transformers
A small demo that uses Hugging Face transformers pipeline (DistilBERT) to classify sentiment.

## Files
- `sample_texts.csv` - small example texts
- `sentiment_transformers.py` - example script using transformers pipeline
- `requirements.txt` - dependencies

## How to run
```bash
pip install -r requirements.txt
python sentiment_transformers.py
```
